# 自动内存管理机制
## 运行时数据区域
Java 虚拟机在执行 Java 程序的过程中会把它所管理的内存划分为若干个不同的数据区域.
* 方法区(Method Area): 由所有线程共享的数据区
* 堆(Heap): 由所有线程共享的数据区
* 虚拟机栈(VM Stack): 线程隔离的数据区
* 本地方法栈(Native Method Stack): 线程隔离的数据区
* 程序计数器(Program Couter Register): 线程隔离的数据区
